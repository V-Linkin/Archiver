# 拾屿 Gatherly Windows 用户指南

---

## 1. 如何运行 Windows 版

1. 下载并解压发布包到任意目录
2. 双击 `Gatherly.Windows.exe` 启动
3. 首次启动会自动创建数据目录和数据库

---

## 2. 如何导入 macOS 备份 zip

### 前提条件

在 macOS 上使用拾屿 Archiver 导出备份 zip 文件，传输到 Windows 电脑。

### 导入步骤

1. 启动 Gatherly Windows
2. 点击左侧 Sidebar 底部的「📦 导入备份」按钮
3. 在文件选择器中选择 macOS 导出的 `.zip` 文件
4. 等待导入完成
5. 导入成功后会显示「导入成功」提示

### 注意事项

- 仅支持恢复到空数据库（首次启动或清空数据后）
- 如需重新导入，需手动删除数据库文件后重启

---

## 3. 数据保存在哪里

所有数据保存在本地：

```text
%LOCALAPPDATA%\Gatherly\
```

在文件资源管理器地址栏输入 `%LOCALAPPDATA%\Gatherly` 即可访问。

### 目录结构

```text
%LOCALAPPDATA%\Gatherly\
├── Gatherly.db          ← SQLite 数据库
├── Gatherly.db-shm      ← 共享内存文件（自动生成）
├── Gatherly.db-wal      ← WAL 日志文件（自动生成）
├── media\               ← 图片和视频文件
│   ├── {item-uuid}\
│   │   ├── cover.jpg
│   │   ├── image_001.jpg
│   │   └── video.mp4
│   └── ...
└── platform_logos\      ← 平台 Logo 文件
    └── {logo-uuid}.jpg
```

---

## 4. 图片 / 视频文件在哪里

从 macOS 备份导入后，图片和视频文件位于：

```text
%LOCALAPPDATA%\Gatherly\media\
```

每个内容的媒体文件存储在以其 UUID 命名的子目录中。

---

## 5. 如何搜索

1. 点击左侧 Sidebar 的「🔍 搜索」
2. 在搜索框中输入关键词
3. 按 Enter 或点击「搜索」按钮
4. 搜索结果以卡片形式展示
5. 点击卡片查看详情

---

## 6. 如何按平台浏览

1. 左侧 Sidebar 显示平台列表
2. 点击任意平台名称进入该平台内容页
3. 内容以网格卡片形式展示
4. 「未分类内容」显示没有平台归属的内容

---

## 7. 如何查看详情

1. 在首页、平台页或搜索结果中点击任意卡片
2. 进入完整详情页
3. 详情页显示：标题、平台、作者、时间、图片、视频、正文、备注、链接
4. 点击左上角「← 返回」回到上一页

---

## 8. 如何打开图片查看器

1. 进入详情页
2. 在图片区域点击任意图片
3. 弹出图片查看器窗口
4. 点击 ◀ ▶ 或按键盘左右键切换图片
5. 按 ESC 关闭查看器

---

## 9. 如何打开视频

1. 进入详情页
2. 在视频区域找到视频文件
3. 点击「打开视频」用系统默认播放器播放
4. 点击「打开所在文件夹」在资源管理器中定位文件

---

## 10. 如何使用回收站

### 移入回收站

1. 进入详情页
2. 点击右上角「移入回收站」按钮
3. 确认后内容移入回收站

### 查看回收站

1. 点击左侧 Sidebar 的「🗑️ 回收站」
2. 查看已删除的内容列表
3. 每条内容显示剩余天数

### 恢复内容

1. 在回收站中点击要恢复的内容
2. 点击「恢复」按钮
3. 内容回到原位置

### 彻底删除

1. 在回收站中点击要删除的内容
2. 点击「永久删除」按钮
3. 确认后内容被永久删除
