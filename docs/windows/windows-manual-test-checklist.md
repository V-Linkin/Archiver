# Windows MVP 手动测试清单

> 本清单用于人工逐项验证 Windows MVP 功能。
> 每项测试后在 `[ ]` 中填入 `[x]` 表示通过。

---

## A. 启动测试

- [ ] App 能正常启动，无崩溃
- [ ] MainWindow 显示三栏布局（左侧 Sidebar / 中间内容 / 右侧详情）
- [ ] Sidebar 显示"拾屿"标题
- [ ] Sidebar 显示"🏠 首页"按钮
- [ ] Sidebar 显示"🔍 搜索"按钮
- [ ] Sidebar 显示"🗑️ 回收站"按钮
- [ ] Sidebar 底部显示"📦 导入备份"按钮
- [ ] 点击各导航按钮能正常切换（即使当前无数据）

---

## B. 数据库测试

- [ ] 首次启动能自动创建数据库
- [ ] 数据库文件位于 `%LOCALAPPDATA%\Gatherly\Gatherly.db`
- [ ] 数据目录位于 `%LOCALAPPDATA%\Gatherly\`
- [ ] 重启应用后数据仍存在（不丢失）

---

## C. macOS 备份恢复测试

前置条件：准备一个 macOS 导出的 zip 备份包（包含 `archiver.db`、`media/`、`platform_logos/`）

### C1. 正常恢复

- [ ] 点击"📦 导入备份"按钮
- [ ] 弹出文件选择器，仅显示 `.zip` 文件
- [ ] 选择 zip 文件后开始导入
- [ ] 导入过程中显示"正在导入..."状态
- [ ] 导入成功后显示"导入成功"状态（绿色）
- [ ] 首页出现恢复的数据
- [ ] 搜索能搜到恢复的数据
- [ ] 详情页能显示恢复的数据（标题/平台/作者/正文）
- [ ] `media/` 文件被复制到 `%LOCALAPPDATA%\Gatherly\media\`
- [ ] `platform_logos/` 文件被复制到 `%LOCALAPPDATA%\Gatherly\platform_logos\`

### C2. 异常处理

- [ ] 选择非 zip 文件时，文件选择器不显示该文件
- [ ] 取消文件选择时不报错，不显示任何状态
- [ ] 对非空数据库恢复时，显示"导入失败"错误信息（红色）
- [ ] 选择不存在的文件时，显示"导入失败"错误信息

---

## D. 首页 / 列表测试

前置条件：数据库中已有数据（通过备份恢复或直接操作）

- [ ] 首页加载最近导入的内容列表
- [ ] 内容标题正确显示（空标题显示"未命名内容"）
- [ ] 平台标签正确显示（如 bilibili、youtube）
- [ ] 作者正确显示（空作者显示"未知作者"）
- [ ] 导入时间正确显示
- [ ] 点击列表中的 item 后，右侧详情区更新为该 item 的信息

---

## E. 搜索测试

- [ ] 输入英文关键词后点击搜索，显示匹配结果
- [ ] 输入中文关键词后点击搜索，显示匹配结果
- [ ] 空关键词搜索不报错，结果列表为空
- [ ] 搜索结果可选中
- [ ] 选中搜索结果后，右侧详情更新
- [ ] 已删除（回收站中）的内容不出现在普通搜索结果中

---

## F. 详情页测试

### F1. 未选择 item

- [ ] 右侧显示"选择一个内容以查看详情"占位文字
- [ ] 显示 📄 图标

### F2. 已选择 item

- [ ] 标题正确显示（大号加粗）
- [ ] 平台标签正确显示（蓝色背景）
- [ ] 作者正确显示
- [ ] 发布时间正确显示
- [ ] 导入时间正确显示
- [ ] 正文内容正确显示
- [ ] 备注区域正确显示
- [ ] 原始 URL 正确显示（带下划线）
- [ ] 标准化 URL 正确显示（灰色）
- [ ] 底部显示"移入回收站"按钮

---

## G. 备注编辑测试

- [ ] 点击"编辑备注"按钮进入编辑模式
- [ ] 编辑模式显示 TextBox + "保存" + "取消"按钮
- [ ] 输入备注后点击"保存"，备注更新显示
- [ ] 点击"取消"不保存修改，恢复原备注
- [ ] 清空备注后保存，备注显示为空
- [ ] 重启应用后备注仍存在（持久化验证）

---

## H. 回收站测试

### H1. 移入回收站

- [ ] 在详情页点击"移入回收站"按钮
- [ ] item 从首页列表中消失
- [ ] 回收站列表中出现该 item

### H2. 恢复

- [ ] 在回收站中选中一个 item
- [ ] 右侧详情区显示该 item 信息
- [ ] 点击"恢复"按钮
- [ ] item 从回收站列表中消失
- [ ] 首页列表中重新出现该 item

### H3. 永久删除

- [ ] 在回收站中选中一个 item
- [ ] 点击"永久删除"按钮
- [ ] item 从回收站列表中消失
- [ ] 首页列表中不出现该 item
- [ ] 搜索不到该 item

## I. 不支持功能确认

以下功能当前不支持，确认不报错即可：

- [ ] 不测试 Parser（无自动抓取功能）
- [ ] 不测试 WebView2（无浏览器嵌入）
- [ ] 不测试安装包（直接 `dotnet run` 启动）
- [ ] 不测试自动更新（无更新功能）
- [ ] 不测试备份导出（仅支持恢复）
- [ ] 不测试批量操作（仅单条操作）
- [ ] 不测试新建内容（仅查看已有数据）
- [ ] 不测试标题/正文编辑（仅支持备注编辑）
- [ ] 不测试移动文件夹/移动平台（未实现）
